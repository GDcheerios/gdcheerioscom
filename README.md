## Wanna do stuff?
make a .env file like this:
```txt
DOMAIN = "http://127.0.0.1"
PORT = 8080
SECRET = "MySecret"
IS_PRODUCTION = 0 OR 1  # TRUE | FALSE

SMTP_HOST = "smtp.gmail.com"
SMTP_EMAIL = "no-reply@domain.com"
SMTP_PASSWORD = "password"

DB_HOSTNAME = "hostname"
DB_PASSWORD = "password"
DB_USER = "user"
DB = "database"

OSU_API_KEY = "apikey"
OSU_SECRET = "secret"
CLIENT_ID = 1234
```
